# ImGuiRDR2Hook
 A Vulkan & DirectX 12 hook that allows you to create ImGui menus for RDR2
 
# Prerequisites
- [MinHook x64](https://github.com/TsudaKageyu/minhook)
- [Kiero](https://github.com/Rebzzel/kiero)
- [Vulkan SDK](https://vulkan.lunarg.com/sdk/home)
